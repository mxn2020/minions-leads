# Minions Leads

Lead records, company profiles, contact details, and qualification status

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-leads/sdk`) — Core library for Node.js
- **Python SDK** (`minions-leads`) — Python bindings
- **CLI Tool** (`@minions-leads/cli`) — Command-line interface
- **Web App** — Interactive playground at [leads.minions.wtf](https://leads.minions.wtf)
